## PORTIFÓLIO FEITO PARA MOSTRAR MEU PROGRESSO NOS ESTUDOS

## 1. Gestão de contatos
Este é um projeto prático desenvolvido para treinar operações de **CRUD** (Create, Read, Update, Delete) e manipulação de arquivos externos utilizando **Python**.
##  Funcionalidades
O sistema permite gerenciar uma agenda de contatos diretamente pelo terminal, com as seguintes operações:

1.  **Adicionar Contato**: Registra Nome, Telefone e E-mail.
2.  **Listar Contatos**: Exibe todos os contatos salvos.
3.  **Editar Contato**: Altera informações de um contato existente buscando pelo nome.
4.  **Excluir Contato**: Remove um registro da lista.
5.  **Persistência de Dados**: Todos os dados são salvos automaticamente em um arquivo `contatos.csv` (compatível com Excel), garantindo que as informações não sejam perdidas ao fechar o programa.
##  Tecnologias Utilizadas

* **Python 3.x**
* **Biblioteca `csv`**: Para manipulação da base de dados.
* **Biblioteca `os`**: Para interação com o sistema operacional (verificação de arquivos e limpeza de console).
##  Conceitos de Programação Aplicados

Este projeto consolida conhecimentos fundamentais de lógica e desenvolvimento:
* **Modularização**: Uso de funções específicas para cada tarefa.
* **Estruturas de Dados**: Manipulação de listas e dicionários.
* **Estruturas de Repetição**: Loops `while` para o menu e `for` para buscas e listagens.
* **Tratamento de Arquivos**: Leitura e escrita (I/O) em arquivos CSV
